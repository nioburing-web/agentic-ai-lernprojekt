# Skill: Reporting Qualität
# Wann: Immer wenn ein Report generiert wird

## Report Struktur
# Betreff: "NIO Automation Report – [Datum]"

# Abschnitt 1: Buchhalter-Outreach Gestern
# - E-Mails gesendet: X
# - Antworten: X (X%)
# - Interessenten: X
# - Abgelehnt: X

# Abschnitt 2: Sofort-Antwort Gestern
# - Anfragen: X
# - Beantwortet: X
# - Ø Reaktionszeit: X Minuten
# - Schnellste Antwort: X Minuten

# Abschnitt 3: Handlungsbedarf
# - Offene Leads (3+ Tage): Liste der Firmen

## Qualitäts-Regeln
# - Nur Zahlen – kein unnötiger Text
# - Immer Datum im Betreff
# - Handlungsbedarf immer am Ende
# - Maximal 20 Zeilen
