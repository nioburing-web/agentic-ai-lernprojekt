# Skill: E-Mail Klassifizierung
# Wann: Immer wenn eine Antwort klassifiziert wird

## Kategorien

### INTERESSIERT
# Signale: "ja", "gerne", "klingt interessant",
# "können wir mal reden", "mehr erfahren",
# "Termin", "Angebot"

### ABGELEHNT
# Signale: "nein", "kein Interesse", "danke aber",
# "nicht relevant", "bereits versorgt",
# "bitte keine weiteren E-Mails"

### RÜCKFRAGE
# Signale: "was genau", "wie funktioniert",
# "welche Kosten", "Frage:", "?"

### ABWESEND
# Signale: "abwesend", "urlaub", "out of office",
# "bin ab", "zurück am", "vertreten durch"

## Regeln
# - Bei Unsicherheit: RÜCKFRAGE
# - Nur eine Kategorie pro E-Mail
# - Kurze Begründung in einem Satz
