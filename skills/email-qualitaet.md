# Skill: E-Mail Qualität
# Wann benutzen: Immer wenn eine E-Mail generiert wird

## Qualitäts-Regeln
# - Beginne mit: "Sehr geehrte/r [Name],"
# - Keine doppelten Signaturen
# - Keine Tippfehler oder doppelten Buchstaben
# - Maximal 4 Sätze Haupttext
# - Immer Call-to-Action am Ende
# - Ton: professionell aber persönlich

## Pflicht-Signatur
# Mit freundlichen Grüßen
# NIO Automation
# anfragen@nio-automation.de
# nio-automation.de

## Verboten
# - "Sehr geehrte Damen und Herren" (zu unpersönlich)
# - Mehr als eine Frage in der E-Mail
# - Doppelte Wörter durch Tippfehler

## Betreff-Format
# "Ihre Anfrage bei NIO Automation – wir melden uns"
